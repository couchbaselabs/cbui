CBUI Row

The <row> element is the basic Flexbox container for elements on a single row, especially multiple elements requiring left AND right horizontal alignment.

  • Rows can and should be stacked as needed and to create pseudo-tables/data grids.
  • Rows reside inside a <section> element — flex item of parent flexbox <main> which governs their gross position on the page relevant to other major page elements.
  • Rows have helper classes for simple modifications to their internal layout rules like align-right & align-top.
