CBUI Row

The <row> element is the basic Flexbox container for elements on a single row, especially multiple elements requiring left AND right horizontal alignment. It replaces the need for floats or tables.

  • Rows can and should be stacked as needed and to create pseudo-tables/data grids.
  • Rows reside inside a <section> element — flex item of parent flexbox <main> which governs their gross position on the page relevant to other major page elements.
  • Rows have helper classes for simple modifications to their internal layout rules like align-right for single item aligned to the right side & align-top for items where the usual vertical alignment of middle is not sufficient.

 

First element auto-aligns left Second element auto-aligns right

HTML Markup

  <row>
  <span>First element auto-aligns left</span>
  <span>Second element auto-aligns right</span>
  </row>

CSS

  row {
      padding: .4rem 0;
      display: flex;
      min-height: 1.5rem;
      align-items: center;
      flex-wrap: no-wrap;
      justify-content: space-between;
  }

How about a collection of form fields?

HTML Markup

  <row>
  <label>Field Number One 
  <input type="text"></label>
  <label>Field Number Two 
  <input type="text"></label>
  </row>
  <row>
  <label>Field Number Three 
  <input type="text" placeholder="I'm a placeholder"></label>
  <label>Field Number Four 
  <input type="text"></label>
  </row>
  <row class="cbui-align-left">
  <label class="cbui-cursor-pointer">
  <input type="checkbox"> Checking me!</label>
  <label class="cbui-cursor-pointer">
  <input type="checkbox"> Noog, check me!</label>
  <label class="cbui-cursor-pointer">
  <input type="checkbox"> I'm the best!</label>
  </row>

CSS

  input[type="text"] {
  display: block;
  }
  
  label {
  margin-right:1.5rem;
  }
  label.cbui-cursor-pointer {
  cursor:pointer;
  }
  input[type="checkbox"], input[type="radio"] {
  cursor:pointer;
  height:.9rem;
  }
